# Malaria Outbreak Prediction & Environmental Risk Analysis

## Project Overview

Public health analytics project focused on understanding the relationship between environmental conditions and malaria outbreaks using IoT-generated environmental data and epidemiological health records from South-South Nigeria.

The project leverages Excel, SQL, Python, and Power BI to analyze outbreak patterns, identify environmental risk factors, and support data-driven public health interventions.

---

## Business Problem

Malaria remains a major public health challenge in Nigeria, contributing significantly to illness and mortality.

Environmental conditions often influence mosquito breeding and disease transmission. Understanding these relationships can help healthcare organizations and public health authorities improve outbreak monitoring and intervention planning.

This project aims to identify environmental and epidemiological factors associated with malaria outbreaks and develop analytical tools to support public health decision-making.

## Dataset Information

### Dataset Name

Malaria Outbreak Dataset

### Source

### Dataset Description

This dataset was collected from the South-South region of Nigeria and contains environmental data captured through IoT devices alongside epidemiological data related to malaria outbreaks.

The dataset combines environmental monitoring information with health outcome data to support outbreak analysis and prediction.
